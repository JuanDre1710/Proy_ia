# IAJuan

`IAJuan` es un proyecto de inteligencia artificial que utiliza un modelo de clasificación para analizar operaciones y detectar posibles fraudes. Este proyecto incluye la simulación de un modelo de aprendizaje automático, la obtención de datos desde una API simulada, y el procesamiento de datos para realizar predicciones.

## Características

- **Entrenamiento de un modelo de ejemplo**: Entrena un modelo de clasificación utilizando un dataset simulado.
- **Carga de un modelo preentrenado**: Carga un modelo previamente entrenado desde un archivo.
- **Obtención de datos desde una API simulada**: Simula la respuesta de una API para obtener datos de entrada.
- **Procesamiento de datos**: Convierte los datos obtenidos en un formato adecuado para el modelo.
- **Predicción de fraude**: Utiliza el modelo para predecir si una operación es normal o fraudulenta.

## Requisitos

- Python 3.9 o superior.
- Librerías necesarias (instalables con `pip`):
  - `pandas`
  - `joblib`
  - `scikit-learn`
  - `requests`
